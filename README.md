# cbridge_robot

## 一、cbridge2.0_test_robot

基于python的cbridge2.0测试网活动机器人
- python3、infra（以太坊节点基础设施）、web3.py（以太坊交互python类库）

实现功能：
- 跨链
- 添加流动性
- sgn 代币stake
- 统计当天分数
- 针对cbridge2.0测试网API限流优化

## 二、layer2_finance_test_robot

基于python的layer2 finance测试网活动机器人
- python3、infra（以太坊节点基础设施）、web3.py（以太坊交互python类库）

自动化实现利益最大化：

1.计算收益率最大的协议

2.计算需要赎回资金的协议

3.从协议赎回资金

4.充值到收益率最大协议中
